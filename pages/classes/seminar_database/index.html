データベース演習

授業科目区分： 専門教育科目
単位数: 1単位
開講学科: 文化情報学科
学年: 3年次
関連科目: データベースシステム
授業テーマ: 関係データベースを利用したウェブサイトの構築

授業内容

2年次に開講される「データベースシステム」の内容を踏まえ， 関係データベースとウェブサイトを連携して運用する方法を学ぶ．ウ ェブサーバにはPythonを採用し，CGI（Common Gate Interface）を利用してデータベースにアクセスする． また，オンライン上のデータベースにアクセス可能なSPARQLと呼ばれる問い合わせ言語についても学習する．

授業計画

  1. データベースとウェブサイトの連携
  2. SQLiteデータベースの作成
  3. Pythonでウェブサーバの構築
  4. ウェブサイトでSQL（１）・データベースの表示
  5. ウェブサイトでSQL（２）・ログインページの作成
  6. ウェブサイトでSQL（３）・ユーザ情報の検索
  7. ウェブサイトでSQL（４）・商品情報の更新
  8. ウェブサイトでSQL（５）・商品管理ページの追加
  9. ジャパンサーチで検索（１）・簡易Web API
  10. ジャパンサーチで検索（２）・SPARQL
  11. Linked Open Data（１）・DBPedia
  12. Linked Open Data（２）・統計LOD
