**科目種別:** 講義  
**テーマ:** プログラムの基本的な構造の理解とコードの実装  

### 概要

アプレット、組み込みプログラム、Webアプリケーションなどに広く利用されているプログラミング言語「Java」の基本を学習する。演算子や制御構文などプログラムの基本構造の修得から始め、ゲームなどの応用的なプログラムに発展させる。各回に出題される課題をこなしながら、目的に合わせたコードを実装する能力を養う。
