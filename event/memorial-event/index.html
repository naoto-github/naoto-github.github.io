椙山女学園大学 文化情報学部

２０周年記念行事


椙山女学園大学・文化情報学部は，2019年度に開設20周年を迎え，
3,800名を越える卒業生を輩出してきました．
記念すべき20周年を祝い，学部同窓会が記念パーティを，開催する運びとなりました
ご多忙とは存じますが，懐かしいひとときを過ごして頂きたく，
皆様でお誘い合わせの上，記念パーティにご参加ください．

記念アルバムの制作に関して

思いでの写真提供のお願い

記念パーティに合わせ，余興の一環として，卒業生の思いでの写真を集めた記念アルバムを制作することになりました．そこで，卒業生と学部教育に携わった教員の皆様を対象に，思いでの詰まったお写真のご提供をお願い申し上げます．ご提供にご協力頂ける場合は，別途，送信用フォームのURLをお伝えしますので，７月２０日（土）までに同窓会事務局までご連絡ください（ご提供頂いたお写真は記念アルバムのみに使用し，その他の目的で使用することはありません）．

椙山女学園同窓会事務局（月～金10:00～16:00）
TEL: 052-781-5952 FAX: 052-781-7197
E-mail: itogiku@og.sugiyama-u.ac.jp

OG講演会

20周年記念行事のプレイベントとしてOG講演会を
2018年10月21日に実施しました．
パーソルR＆D株式会社の林美紗子さん（平成26年度卒）をお招きし，
IT業界の裏側や人材育成に関してご講演頂きました．

OG講演会の詳細をチェック！

OGと現役学生の座談会

20周年記念行事のプレイベントとして「OGと現役学生の座談会」を
2019年1月22日に実施しました．
服部佳那さん（平成27年度卒）と柴山智香さん（平成28年度卒）をお招きし，
４人の現役学生と一緒に”学部での学び”や”卒業後のキャリア”に関して
和気あいあいとした雰囲気の中で話し合いました．
